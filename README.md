# Netflix Machine Learning Classification Project

## Project Overview
This project predicts whether a Netflix title is a Movie or a TV Show using a Decision Tree Classifier.

# Dataset
- Netflix Titles Dataset

# Technologies Used
- Python
- Pandas
- Scikit-learn
- Jupyter Notebook (Kaggle)

# Machine Learning Workflow
1. Load Dataset
2. Explore Data
3. Clean Data
4. Select Features
5. Label Encoding
6. Train-Test Split
7. Decision Tree Classifier
8. Model Prediction
9. Accuracy Evaluation
10. Confusion Matrix

# Features Used
- Release Year
- Duration
- Rating
- Listed In (Genre)

#Target
- Type (Movie / TV Show)

#Algorithm
- Decision Tree Classifier

# Result
- Accuracy: **99%**
